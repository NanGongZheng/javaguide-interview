## 🚀 本地运行

```bash
# 方式一：Python
python3 -m http.server 8080

# 方式二：Node.js
npx serve .
```

然后访问 http://localhost:8080

## 🌐 GitHub Pages 部署

1. 推送到 GitHub 仓库
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，目录选 `/ (root)`
4. 保存后即可通过 `https://<username>.github.io/<repo-name>/` 访问
